# README

This is sample api to users manager.

* Ruby version 2.7.0

* Rails version 6.0.3.4

* Configuration
- Postgresql
- Devise Token Auth
- Pundit
- Rspec

* Database creation

    ```
    rails db:create
    ```

* Database initialization
    ```
    rails db:migrate
    rails db:seed
    ```

* How to run the test suite
    ```
    bundle exec rspec spec
    ```
